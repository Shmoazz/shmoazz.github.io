5 DOF Robotic Arm (In Progress)

Ongoing Project, 2025 Back to Projects

Project Overview

I just started this project and have no physical prototype yet. The goal is to create a fully 3D printed 5 DOF robotic arm. It will champion ease of assembly, and will serve as a tool to help me start to practice writing control and motion planning algorithms for serial robotic arms. As of right now it's just the start of a CAD model, but I'm excited about its potential applications once completed.

Planned Technologies

  • 3D printing for all structural components
  • Servo motors for joint actuation
  • Raspberry Pi 5 as the main controller
  • Control software still to be determined
  • Forward and inverse kinematics implementation

Project Goals

When completed, this robotic arm will serve as both a learning platform and a practical tool. I aim to implement various motion planning algorithms, explore different control approaches, and potentially integrate it with computer vision for autonomous object manipulation. The modular design will allow for future upgrades and modifications as I continue to learn and experiment. Specifically I hope to employ skills learned from my robotics and automation class, where I was taught the foundation for serial robotic arm control.
